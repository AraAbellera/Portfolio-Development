SOI Learning Portal

Skills Used: HTML, CSS, Bootstrap, Java, JavaScript, VSCode

Description:
This is website about a learning portal that provides IT classes. The main page contains multiple information such as courses, benefits, and direct links to other pages.

How I made it:
First, I added the necessary scripts (Bootstrap CSS, Bootstrap JS, jQuery) and then linked my external stylesheet to my HTML using (link rel="stylesheet" href="stylesheet.css"). Next, I added a navigation bar using (nav class="navbar navbar-expand-lg navbar-light bg-light"). Next, I added containers for the Banner and Three-column sections using (div class="container-fluid" id="banner-fluid")(div class="container-lg" id="banner-lg")(div class="row justify-content-md-center").
