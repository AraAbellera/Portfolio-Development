Modules App

Skills Used: Android Studio, Java

Description:
This is an android app that contains modules and their information such as the module code, name, academic year, semester, module credit, venue, and lecturer. This app makes it easier for students to view their module information without needing to open their laptop and go to LEO.

How I made it:
I made two Java classes named “MainActivity.java” and “ModuleDetailActivity.java”.
In the “MainActivity.java” class, I added a function to each module using setOnClickListener(). Inside the setOnClickListener(), I used Intent and startActivitty button so that when pressed it goes to another page containing more information about that module. I also used putExtra() to add the different names and values to each button.
Next, in the “MainActivity.java” class, I used getIntent() and getStringExtra() to receive the request to open the page containing more information about the modules. Next, I used and IF ELSE statement and equals() for the individual buttons so that they show different information in the page. In the IF ELSE statement, I add text to the page using setText()
