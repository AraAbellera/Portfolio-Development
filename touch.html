TOUCH Services App

This is an app that teaches the elderly mapping skills that can help them in their everyday lives. The app contains 3 skills that can be learned which are COmpass, Morse Code, and Directions. In the Compass page, the user learns about East West North South and has to answer simple quizzes to pass. In the Morse Code page, the user is required to tap the buttons based on the morse code given and in the Directions page the user is required to physically go to a location. Furthermore, users can ask questions in the questions page if they are unsure about anything and the app can be audible if the user has sight problems
