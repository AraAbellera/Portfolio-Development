Bill Please!

Skills Used: Android Studio

Description:
This is an android app that helps users calculate how much they need to pay. It allows users to add the total amount, discount and toggle gst and service charge if there are any which makes it easier for people to calculate the total amount.

How I made it:
I added a function to the “Split” button by using setOnClickListener. Next, I get the String value of the “Amount”, “Number of Pax”, “Discount” fields using getText() and toString(), then I changed it to number by using Integer.parseInt() . Next, I used an IF ELSE statement and isChecked() to add the individual equations for the “SVC” and “GST” buttons depending on which button is pressed. Next, I used and IF ELSE statement to check which payment method is chosen and display the different text for each payment method. Next, I added the final value in the Total Bill section by using setText(String.format()) For the “Reset” button, I added a function by also using setOnClickListener. Next, I made all the fields blank by using setText()
